# KLoROS Deus Ex Machina v2 (Add‑On Pack)

This add‑on gives you:
- **RAG Quality D‑REAM experiment** (keeps best passing fusion weights)
- **Metrics logger** (voice + RAG) and a **dashboard** script
- Optional **systemd** unit for scheduled experiments (disabled by default)

Install:
  bash install.sh

Run:
  python /home/kloros/dream/run_rag_quality.py
  bash /home/kloros/scripts/metrics.sh
  python /home/kloros/scripts/metrics_dashboard.py
